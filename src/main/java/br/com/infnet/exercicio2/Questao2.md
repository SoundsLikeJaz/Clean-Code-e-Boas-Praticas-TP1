# Tarefa: Explique como este código falha em atingir os quatro objetivos de um código de alta qualidade:
- Legibilidade;
- Redução de duplicação;
- Facilidade de modificação;
- Eficácia na comunicação.

O código falha em atender aos quatro objetivos de qualidade principalmente pelo aninhamento excessivo de 
condicionais. Essa estrutura reduz a legibilidade e repete verificações desnecessárias, comprometendo a clareza e a 
simplicidade. Além disso, dificulta futuras modificações, já que novas regras exigiriam mais níveis de decisão. Por 
fim, o fluxo pouco natural prejudica a comunicação do código, tornando sua intenção menos evidente para outros 
desenvolvedores.
